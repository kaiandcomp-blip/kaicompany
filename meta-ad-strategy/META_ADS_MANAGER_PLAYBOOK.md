## 🎬 광고 1: 진단 리포트 (주력)

**광고명:** `AD_01_VID_진단_Form`

### 텍스트

**주 헤드라인 (메인):**
```
"마케팅은 수수료 전투, 성과는 제자리?
카이앤컴퍼니의 무료 진단을 받아보세요"
```

**보조 텍스트:**
```
3가지만 입력하면 실제 광고 전략부터 예상 매출까지 
한눈에 보는 '30분 전략 보고서'를 무료로 받아봅시다.

✓ 현재 채널별 ROAS 진단
✓ 매출 2배 달성 시나리오
✓ 다음달 구체적 액션플랜

기획+개발+광고 원팀이 직접 분석합니다.
```

**설명 텍스트:**
```
"150개 이상 e커머스 브랜드를 성장시킨 원팀의 노하우.
매출을 극대화하는 데이터 기반 전략을 지금 무료로 확인하세요."
```

**CTA 버튼:**
```
텍스트: 무료 진단받기
목적지: kaicompany.kr/consult (또는 문의 폼)
```

### 소재

**영상 요구사항:**
- 포맷: 세로(1080×1920px 또는 9:16 비율)
- 길이: 15초
- 콘텐츠:
  - 0-5초: 문제 제기 ("수수료 높아졌나?", "매출 정체?")
  - 5-10초: 진단 가치 ("3분 입력→30분 보고서")
  - 10-15초: CTA ("지금 무료진단 받기")
- 자막: 한글(자동 또는 수동)
- 음성: BGM (저작권 자유)

**또는 이미지 (정적):**
- 포맷: 정사각형(1200×1200px)
- 배경: 카이 주황(#ff5f1f) 또는 중립톤(회색)
- 텍스트: "무료 전략 진단" + 로고
- CTA: 명확한 버튼 형태
