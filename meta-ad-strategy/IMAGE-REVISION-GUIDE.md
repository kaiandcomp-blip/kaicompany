## 🔴 Image 2: 케이스스터디 (수정 필수!)

### 현재 상태 ⚠️ **오류 발견**
```
좌측: 일반 텍스트
"김은 2번인데
매출은 4.2배 달렸습니다"

우측: 차트 + 수치
2025.02: 25,100,000원
2026.02: 105,553,640원
상단: 42.2배 증가 (❌ 잘못됨!)
```

### 수정안 ✅
```
좌측: 일반 텍스트 [변경 없음]
"김은 2번인데
매출은 4.2배 달렸습니다"

우측: 차트 + 수치 [수정됨]
2025.02: 25,100,000원
2026.02: 105,553,640원

⭐ 수정 포인트:
- "42.2배 증가" → "4.2배 증가" 또는 "+320%" 변경
- 오렌지 배지에 명확히 표시

추가 정보 (아래 나타내기):
"3개월 운영 기준 / 식품류 셀러"
```

### 계산 검증
```
105,553,640 ÷ 25,100,000 = 4.206... ≈ 4.2배

현재 표기: 42.2배 ❌
수정: 4.2배 ✅ (또는 320% 증가)
```

### 수정 상세
**1️⃣ 수치 정정 (필수)**
- 위치: 오렌지 원형 배지 중앙
- 현재: "42.2배 증가"
- 변경: "4.2배 증가" 또는 "+320%"
- 폰트 크기: 조금 더 크게 (가독성)

**2️⃣ 추가 정보 (신뢰도)**
- 위치: 차트 아래
- 내용: "3개월 운영 기준 / 식품류 셀러"
- 글씨: 작은 회색
- 목적: 신뢰도 확보 ("모든 업체는 아니고...")

**3️⃣ CTA 추가 (선택)**
- 위치: 우측 하단
- 내용: "당신의 성과도 계산해보세요"
- 버튼: "ROI 계산기" (클릭 가능하면)

## 🟡 Image 3: 포트폴리오 (가독성 개선)

### 현재 상태
```
12개 에뜨 (3x4 그리드)
각 에뜨 상단: 카테고리 이름
각 에뜨 중앙: 상품 사진
각 에뜨 하단: 성과/메시지 텍스트

헤더: "기획부터 디자인, 세팅까지 ROAS 중심으로 만들어드립니다"

풋터: "광고비 쓰는데 매출이 안 오른다면,
콘텐츠부터 바꿔야 합니다.
브랜드는 해치지 않고 구매 욕구는 더 강하게."
```

### 수정안 ✅
```
구조 변경: 12개 → 6개 선택 (가독성 극대화)

추천 6개 (다양성 유지):
1. 방울토마토 (농산물) - 감정 Hook 강함
2. 굴 신기술 (해산물) - 비포/애프터 명확
3. 3단 원대 세팅 (가성비) - 가격 포인트
4. 라면 (유명도) - 브랜드 파워
5. 홈카페 (생활용품) - 라이프스타일
6. 음식점 (B2B) - 업종 다양성

레이아웃: 2x3 (세로 3개, 가로 2개)

각 에뜨 개선:
- 텍스트 축약 (현재: 장문 → 수정: 1~2줄)
- 핵심 수치만 (예: "4.2배" 또는 "+320%")
- 아이콘 추가 (📈 증가, 💰 가격, ⭐ 인기도)

헤더: [유지] "기획부터 디자인, 세팅까지 ROAS 중심으로 만들어드립니다"

풋터: [약간 수정]
"광고비 쓰는데 매출이 안 오른다면,
콘텐츠부터 바꿔야 합니다.
⭐ 카이는 브랜드를 해치지 않고 구매욕구를 강하게."

CTA 추가:
"당신의 사례도 만들어보세요 → 무료 진단 신청"
```

### 수정 상세
**1️⃣ 에뜨 선택 (12개 → 6개)**

현재 12개 중 강도가 낮은 것 제거:
- ❌ 제거: 노선수자동화, 대구 어족, 이벤트 음식, 에어로웨이 픽
- ✅ 유지: 방울토마토, 굴, 가성비 세팅, 라면, 홈카페, 음식점

**2️⃣ 텍스트 축약**

예시 변경:
```
현재 (길음):
"제 밭공 보고 도서세요
도상로 공사 참봉 보고 될 도세밖니다"

변경 (짧음):
"신선한 농산물 4.2배 증가"
"⭐ 공급자 직접 인증"
```

**3️⃣ 아이콘 활용**
- 📈: 성장 (ROAS 4.2배)
- 💰: 가격 (3만원 vs 8만원)
- 👥: 인기도 (월 500건+)
- ⭐: 신뢰도 (실제 사례)

**4️⃣ 레이아웃 변경**
- 현재: 3x4 (12개, 복잡함)
- 변경: 2x3 (6개, 명확함)
- 각 카드 크기: 조금 더 큼 (가독성)
