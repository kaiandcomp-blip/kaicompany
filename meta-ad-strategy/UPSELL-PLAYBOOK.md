## 0. 경영진 요약 (1페이지)

**한 줄 전략**
> 유지보수 고객의 공통 불만("소재는 좋은데 성과가 안 난다")을, 카이가 비워둔 **메타광고(인스타·페이스북)** 영역으로 전환해, **원팀·실시간 투명 리포트·24시간 소재 교체**라는 검증 가능한 무기로 추가 계약을 따낸다. 텍스트 약속이 아니라 **실제 측정 대시보드(02)·소재 시스템(03)·캠페인 구조(04)를 산출물로 보여주며** 판다.

**타겟 우선순위 (Who)**
- **Tier 1 (즉시 업셀, 최우선)**: 이미 광고비 지출 중·성과 불만·월 매출 1,000만+·갱신 시점. 10~15개사 [가정]. → 직접 전화 + 무료 진단.
- **Tier 2 (중기 업셀)**: 소규모 광고비·성과 측정 부재·월 매출 300~1,000만. 20~30개사 [가정]. → 이메일 + 소셜 리타겟 + 재상담.
- **Tier 3 (신규 유치)**: 유지보수 미계약. → **`STRATEGY.md` 자사 리드젠 캠페인(S2 무료진단 오퍼)이 그대로 유입 엔진.** 본 문서에서 중복 설계하지 않고 연계만 한다.

**상품·가격 (What) — 운영비(카이 수익)와 광고비(클라이언트 부담)를 반드시 분리**

| 옵션 | 카이 운영비(월, =증분 MRR) | 광고비(클라 부담, 별도) | 클라 월 총지출 | per-client 목표 ROAS* | 추천 |
|---|---|---|---|---|---|
| **A 메타 단독** | ₩1,500,000 | ₩1,500,000~3,000,000 | ₩3,000,000~4,500,000 | 4~6x | 기획·디자인 외부 보유 셀러 |
| **B 통합 패키지** | ₩2,000,000~2,500,000 (기존 100만 + 메타 100~150만) | ₩1,500,000~3,000,000 | ₩3,500,000~5,500,000 | 5~7x | 전문 대행 원하는 성장형 셀러 |
| **C 성과기반** | 기본 ₩1,000,000 + 광고비의 15% | ₩2,000,000~3,000,000 | ₩3,300,000~4,450,000 | CPA 우선 | 호갱 경험·성과 중시 셀러 |

> *목표 ROAS는 **레인지(보장 아님)**. "평균 ROAS 6.8x"는 카이 클라이언트 평균 실적(브랜드 자산)이며 광고/제안서 노출 시 **항상 "달성 조건(예산·기간·업종) 공개" 별주**와 함께 쓴다(메타 정책상 "보장/무조건/N배 확정" 표현 금지 — 03 가드레일).

**핵심 KPI 3개 + 목표 (가정 — 운영 1개월 후 실측 재보정)**

| KPI | 보수 | 기본(목표) | 공격 | 정의 |
|---|---|---|---|---|
| **Month 1 신규 업셀 계약** | 2건 | **3건** | 4건 | Tier1·2 합산 신규 메타 계약 |
| **증분 MRR(카이 운영 수익)** | +₩3,000,000 | **+₩4,500,000** | +₩6,000,000 | 신규 계약 × 평균 운영비 ₩150만(광고비 제외) |
| **업셀 전환율(제안→계약)** | 30% | **40%** | 50% | Tier1 기존 신뢰 기반(콜드 아님) |

> 누적(가정): Month 2~3 = 6~12건·MRR +₩900~1,800만 / Month 6 = 15~30건·MRR +₩2,250~4,500만. (per-client 운영비 ₩150만 기준. 광고비 대리집행액은 카이 수익이 아니므로 MRR에서 제외 — 6장 정합 노트 #3.)

**지금 당장 실행 3가지**
1. **Tier 분류 + 무료진단 리포트 템플릿 확정.** 유지보수 고객 전수를 Tier 1/2로 분류(접촉 우선순위·담당자 배치). 무료진단 산출물은 **`STRATEGY.md` 0주차의 "진단 리포트 샘플(실제 대시보드)"을 그대로 재사용** — 새로 만들지 않는다.
2. **Tier 1 직접 접점(전화+무료진단) 착수.** 10~15개사에 스크립트(4장) 기반 콜 → 30분 무료 진단 상담 → 맞춤 제안서(8장 템플릿) 3종(A/B/C) 발송.
3. **3종 제안서·견적 + Objection 시트 완성.** ROAS 표기는 전부 "평균 6.8x(조건 공개) + per-client 목표 레인지"로 통일(보장 표현 제거). 운영비/광고비 분리 표기 필수.

## 1. 왜 메타광고 업셀인가 (기회 정의)

**유지보수 패키지의 구조적 빈틈**
```
월 100~110만원 패키지: ①기획/전략 ②디자인/소재(배너 10개 등) ③광고 운영(SNS·쇼핑몰) ④분석/리포팅 ⑤주기 보수
→ 공통 불만: "소재 제작은 좋은데 광고 성과가 안 난다"
```
- 현재 고객은 **기본 운영만** 하고 있고, **메타광고(인스타·페이스북)는 카이가 비워둔 영역**. 여기가 업셀 포인트.
- "성과가 안 난다"의 진짜 원인은 대개 **측정 부재(픽셀/CAPI 없음)·소재 피로 방치·채널 분산**. 이 셋이 카이 USP(원팀·24h 교체·투명 측정)와 정확히 맞물린다.

**판매하는 "성과 역량"의 실체 (영업 약속 ↔ 운영 산출물 정합)**

| 영업에서 말하는 것 | 실제 운영 근거(산출물) | 비고 |
|---|---|---|
| "투명 실시간 리포트" | 02 §6 **3층 대시보드**(Ads Manager ≈ GA4 ≈ CRM), 픽셀+CAPI 설치 | 텍스트 약속이 아니라 화면을 보여줌 |
| "24시간 소재 교체" | 03 §5 로테이션·피로도 신호→액션 표 | frequency·CTR·CPM·CPL 임계 → 24h 내 교체 |
| "원팀(재하청 0)" | 01-E3 정면 반박 메시지, 04-⑪2 퍼널 누수 0 | 기획·개발·광고 in-house |
| "평균 ROAS 6.8x" | 브랜드 자산(클라이언트 e커머스 실적). 달성 조건 병기 | 산업 표준 2~4x[추정] 대비 우위 |
| "정확한 측정으로 성과" | 02 픽셀+CAPI(EMQ 6.0+·디듀프), e커머스는 전환=Purchase | 클라별 사이트에 개별 설치 필요 |

> 핵심: 클라이언트는 e커머스(상품 판매)이므로 **전환 이벤트 = Purchase/AddToCart**이고 **ROAS는 프록시가 아니라 실매출 기반**이다. (02/04의 "Lead·ROAS프록시"는 카이 자사 리드젠 기준 — 클라 캠페인엔 그대로 쓰지 말고 Purchase 기준으로 재설계. 6장 #5.)

| Tier | 특징 | 핵심 메시지 | 액션 | 규모[가정] | 전환율[가정] |
|---|---|---|---|---|---|
| **1 즉시(최우선)** | 이미 광고비 지출 중·성과 불만(CTR↓·CPL↑)·월 매출 1,000만+·갱신 시점 | "네이버·구글만? **메타로 새 시장을. 원팀이 24시간 최적화합니다.**" | 직접 전화 + 30분 무료 진단 → 제안서 | 10~15개사 | 30~50% |
| **2 중기** | 광고비 소규모(월 10~50만)·성과 측정 부재(대시보드 없음)·월 매출 300~1,000만 | "지금은 작아도 **메타로 빠르게. 원팀이 있으니 가능합니다.**" | 이메일 + ROI 계산기 + 소셜 리타겟 → 재상담 | 20~30개사 | 10~20% |
| **3 신규** | 유지보수 미계약·자체/소규모 대행·월 매출 500만+ | "기획+디자인+**메타까지 한 팀에서.** 그래서 카이가 다릅니다." | **`STRATEGY.md` 리드젠 엔진**(S2 무료진단)으로 유입 → 번들 제안 | 미정(인바운드) | 5~10% |

**타겟 스케일 주의 (정합)**: Tier 1의 *현재* 광고비(월 50~100만)와 메타 *권장 시작* 광고비(월 150~300만, 04 학습 임계 충족치)는 다르다. → 제안 시 **"현 광고비를 메타로 옮기는 게 아니라, 신규 채널에 추가 투입해 새 매출을 연다"**는 **성장 전제**로 설명한다(현 예산 유지 + 메타 신규 추가). [확인 필요] 고객별 실제 광고비/매출.

## 3. 업셀 상품·가격 설계 (What to sell)

> 모든 옵션에서 **운영비(카이 수익, recurring)** 와 **광고비(클라이언트가 메타에 지급, 대리집행, pass-through)** 를 분리해 제시한다. 클라이언트 혼동·내부 MRR 오계산을 막는 핵심 규칙.

### Option A — 메타광고 단독 (운영비 ₩1,500,000/월)
```
구성(운영비 내역):
- 캠페인 설정·관리           ₩500,000  (04 ABO→CBO, S1~S4 구조)
- 크리에이티브 최적화·24h 교체 ₩500,000  (03 훅뱅크·로테이션)
- 실시간 모니터링·KPI 리포팅  ₩500,000  (02 3층 대시보드)
광고비: 별도 ₩1,500,000~3,000,000/월(클라 부담, 대리집행)
per-client 목표 ROAS: 4~6x (레인지·보장 아님)
```
추천: 기획/디자인을 다른 곳에서 이미 하는 셀러.

### Option B — 통합 패키지 (운영비 ₩2,000,000~2,500,000/월)
```
기존 유지보수(₩1,000,000) + 메타광고 강화(₩1,000,000~1,500,000):
- 기획/전략   메타 특화 전략 추가
- 디자인/소재 메타 세로영상 + 정적 이미지 + 캐러셀(03 포맷 세트)
- 메타 운영   전담 운영(04)
- 분석/리포팅 메타 픽셀 기반 실시간 대시보드(02)
광고비: 별도 ₩1,500,000~3,000,000/월(클라 부담)
클라 월 총지출 = 운영 ₩2,000,000~2,500,000 + 광고비 ₩1,500,000~3,000,000 = ₩3,500,000~5,500,000
증분 MRR(카이) = 메타 강화분 ₩1,000,000~1,500,000 (기존 100만은 이미 계상됨)
per-client 목표 ROAS: 5~7x (레인지·보장 아님)
```
추천: 전문 대행사를 원하는 성장형 셀러. (기존 사용자 초안의 "총 아이콘 400~550만"은 *클라 월 총지출* 의미로 정정 — 6장 #6.)

### Option C — 성과 기반 (기본 ₩1,000,000 + 광고비의 15%)
```
기본료: ₩1,000,000 (전략+기획+분석) — 메타 전용 기본료(기존 유지보수와 별도 산정)
성과비: 광고비의 15%
계약 우대: 12개월 계약 시 10% 할인
예) 광고비 ₩2,000,000 → 성과비 ₩300,000 → 총 운영비 ₩1,300,000
예) 광고비 ₩3,000,000 → 성과비 ₩450,000 → 총 운영비 ₩1,450,000
```
추천: CPL/CPA 중시·호갱 경험 업체. (성과비만 광고비 연동분이 카이 수익에 추가됨 — MRR 계산 시 반영.)

**가격 정합 메모**: 평균 증분 MRR/업셀계약 ≈ **₩1,500,000** 가정(A=150만, B 증분 100~150만, C 130~145만의 중앙값). 0장 MRR 목표·6장 시나리오는 이 단가 기준.

## 4. 진행컨섭 프레임 (How to sell) — Tier별 세일즈 모션

### Phase 1 · 타겟팅 & 리스트업 (Day 1~3)
1. 유지보수 계약 고객 전수 추출 → 전체 N개사 확정 [확인 필요, 50~100 가정].
2. Tier 분류: Tier 1(10~15) / Tier 2(20~30) / Tier 3(인바운드는 STRATEGY.md로).
3. Tier별 담당자 배치 + 접촉 우선순위(갱신 임박·불만 이력 우선).

### Phase 2 · 고객 접점 (Day 4~)

**Tier 1 — 직접 전화 + 무료 진단**
- 전화 스크립트:

## 5. Objection Handling (반박 대응 — ROAS 표기 통일 반영)

**Q1. 메타광고는 왜 필요해요?**
> "네이버·구글은 이미 경쟁이 심해요. 메타(인스타·페이스북)는 **다른 고객층**에게 닿습니다 — 발견형 채널이라 신규 유입에 강해요. 기존 채널은 그대로 두고 새 채널을 추가하는 개념입니다."

**Q2. 광고비 추가가 부담돼요.**
> "두 가지를 구분해 드릴게요. **운영비(저희 수수료)** 와 **광고비(메타에 나가는 실예산)** 는 별개예요. 광고비는 작게 시작해 성과 보고 늘리면 됩니다. 저희 클라이언트 **평균 ROAS는 6.8x(달성 조건 공개)** 수준이고, 귀사는 진단 후 **목표 4~7x 레인지**로 현실적으로 잡아드립니다. *보장은 못 하지만*, 새는 곳부터 무료로 찾아드려요." (← 초안의 "ROAS 6.8배 달성" 단정 표현을 정책 안전·신뢰형으로 수정. 6장 #1.)

**Q3. 다른 대행사와 뭐가 다른데요?**
> "①**원팀**: 기획→디자인→광고→분석이 한 팀, 재하청 0. ②**24시간 소재 교체**: 피로해진 소재를 다음날이 아니라 24시간 안에. ③**투명 리포트**: 실시간 대시보드로 어디에 쓰는지 화면으로 보여드려요 — '말뿐인 컨설팅'과 다릅니다."

**Q4. 계약 기간은?**
> "**무료 진단으로 먼저 검증**하고, 만족하시면 최소 3개월로 시작합니다. **12개월 계약 시 10% 할인.**" (1개월 단독 유료 검증은 운영 학습 임계상 성과 판단이 어려워 *무료 진단 + 3개월*로 정정 — 04 학습기 2주·안정 2주 구조. 6장 #7.)
